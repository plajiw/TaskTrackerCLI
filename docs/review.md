# Review de Estudos

## Objetivo

Seguir os requisitos estabelecidos na fonte do projeto [roadmap.sh](https://roadmap.sh/projects/task-tracker). Incluindo, outras features para exercitar lógica de programação, padrões de design e boas práticas.

## Etapas

- Parsing ou análise sintática, processo de analisar e interpretar os argumentos e opções fornecidos pelo usuário ao executar um programa via terminal.

- Divisão da entrada em tokens, o comando é quebrado em partes, como comandos, opções, argumentos posicionais e valores.

- Criação dos Commands atráves do token obtido e usando recursos de Command Pattern;
